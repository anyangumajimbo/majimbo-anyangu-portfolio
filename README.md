# Majimbo Anyangu - Professional Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full-Stack Developer and AI Enthusiast.

## 🌟 Features

- **Responsive Design** - Mobile-first approach with Bootstrap 5.3.2
- **Dark/Light Theme Toggle** - Switch between dark and light modes with persistent storage
- **Dynamic Navbar** - Changes color based on section background for better contrast
- **Animated Hero Section** - Eye-catching animated word rotation with glow effect
- **Smooth Scrolling** - Seamless navigation between sections
- **Project Showcase** - Featured projects with live demos and GitHub links
- **Skills Display** - Visual progress bars and skill categorization
- **Professional Styling** - Custom CSS with CSS variables for easy theming

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3 (with CSS Variables & Animations)
- JavaScript (ES6+)
- Bootstrap 5.3.2
- Font Awesome 6.4.0
- Bootstrap Icons

### Design
- Dancing Script Font for branding
- Custom color scheme with green accent (#40916c)
- Smooth transitions and animations

## 🎨 Customization

### Colors
Edit CSS variables in `assets/css/style.css`:
```css
:root {
    --color-accent: #2d6a4f;
    --color-accent-light: #40916c;
    /* ... more variables ... */
}
```

### Content
Edit `index.html` to update:
- Personal information
- Skills and expertise
- Projects and descriptions
- Education and experience
- Contact details

## 🎯 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/anyangumajimbo/majimbo-anyangu-portfolio.git
   ```

2. Navigate to the project:
   ```bash
   cd majimbo-anyangu-portfolio
   ```

3. Open in your browser:
   - Use VS Code Live Server extension, or
   - Simply open `index.html` in your browser
